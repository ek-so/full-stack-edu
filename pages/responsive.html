Title, 2rem

Units, 1.4rem

  1. Pixels — browser tools to adjust font size don't work
  2. % — relative to the parent element size, has cascading nature.
  3. Em — relative to font-size of the parent, has cascading nature. When applied for width or height, still use font-size as a basis for calculations.
  4. Rem — realive to font-size of the root element (html or just user settings in browser)

Media queries, 1.4rem

  1. For the desktop-first site we use media query with max-width
  2. For the mobile-first, min-width

On mobile all the text becomes bigger, because in media query there's a rule changes <html> font size.
